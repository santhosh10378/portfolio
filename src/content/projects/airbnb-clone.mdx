This is a full-stack Airbnb clone built with React, Node, Express, MongoDB, Tailwind CSS, and Prisma. It replicates the core features of Airbnb, including property listings, booking management, and user authentication.

**Live Website**: [https://hotel-booking-app-frontend-4qp6.onrender.com](https://hotel-booking-app-frontend-4qp6.onrender.com)

## Features

- **Property Listings**: Display available properties in a user-friendly grid.
- **Booking System**: Allow users to book properties for specific dates.
- **User Authentication**: Enable users to sign up, log in, and manage their bookings.
- **Property Management**: Allow property owners to list and manage their properties.

## Technologies

- **React**: A JavaScript library for building user interfaces.
- **Node.js**: A JavaScript runtime for server-side programming.
- **Express**: A web application framework for Node.js.
- **MongoDB**: A NoSQL database for storing property and user data.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Prisma**: An ORM (Object-Relational Mapping) tool to interact with the database.

## Getting Started

To get started with this project, you can clone the repository and install the dependencies:

```bash
git clone
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at [http://localhost:3000](http://localhost:3000).

## Deployment

To deploy this project, you can use a cloud platform that supports Node.js applications, such as Vercel, Heroku, or AWS. You can link your GitHub repository to the platform and deploy the project with a single click.

## Conclusion

This Airbnb clone is a comprehensive example of building a full-stack application with React, Node, Express, MongoDB, Tailwind CSS, and Prisma. By implementing core Airbnb features, you can gain a deeper understanding of full-stack development and create a functional, user-friendly application.
